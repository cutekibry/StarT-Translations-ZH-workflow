# 📖 整合包介绍
## 什么是星际科技？
星际科技是最早的基于 **Minecraft 1.19.2** 版本**格雷科技现代版**的整合包之一，目前已全面升级至 **1.20.1**！虽然目前该整合包仍在开发中，但可完整游玩**至麦克斯韦（*UIV 中期*）阶段**，而 **UHV 之后的部分正在锐意开发中**！

这**不**是一个适合新手的整合包，但我们正在不断改进引导，使其对新玩家更加友好。

## 主要特点
✨ **深入而全面的游戏流程** – 基于格雷科技现代版，带有改变游戏体验的自定义配方和 **UHV** 之后的全新体验！  
🌌 **独特的资源获取方式** – 从**无中生有**开始，进阶到**虚空采矿机**、**原油钻井**，以及复杂但收益丰厚的**产线**。  
🌀 **维度旅行** - 通过**星门**探索宇宙！  
📖 **全面覆盖的任务书** - 随着每次版本更新，为你的旅途提供引导。  
🐱 **小困** - 你忠诚的猫科伙伴，在艰难时期为你提供情感上的支持！

受*全能工厂*、*格雷科技：新视野*、*Technological Journey* 等传奇的**格雷科技整合包**启发，**星际科技**将它们最精彩的创意融入到了**空岛生存模式**中。向它们的创作者致以崇高的敬意——也去看看这些整合包吧！

## 整合包模式
**困难模式** - *挑战更困难的、带有大量新内容的星际科技！*

**阿拜多斯模式** - *在沙漠维度出生！*

要切换实例的整合包模式，只需要打开实例文件夹，并选择一个合适的整合包模式修改器：

* Python 文件适用于所有安装了 Python 的系统
* Shell 文件适用于 Unix、Linux 和 MacOS
* Bat 文件适用于 Windows，但只能在 GitHub 上下载，因为 CurseForge 不允许 .bat 文件

打开整合包模式修改器后，你应当能看到一个询问你要选择 D、H 还是 A 的界面：

* D 是默认模式
* H 是困难模式
* A 是阿拜多斯模式

输入字母即可将实例切换至所选模式。注意不要在游戏运行时切换模式，否则可能会导致文件损坏。
